Сущности:
	Товар: Название, Стоимость, Вес


	Склад
	История перемещений товара

Операции:
	Добавить товар (POST, "/item, ответ: JSON товара, 201 Created" 
	Удалить товар (DELETE, "/item{item_id}", -, 204 No Content
	Получить товар (GET, "/item/{item_id}", ответ: JSON товара, 200 Ok"
	Получить все товары (GET, "/item", ответ: JSON c товарами, 200 Ok"
	Изменить название товара (PATCH, "/item/{item_id}" + JSON с изменяемым полем, ответ: JSON товара, 200 Ok"
	Получить все товары до n кг (GET, "/item?weight=n", ответ: JSON  с товарами, 200 ok"

	Получить все товары на складе
	
	Добавить товар на склад
	Переместить товар между складами
	
	Получить историю перемещений для товара
